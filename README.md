# Restaurant-e
### Conceito do Projeto

Este projeto foi desenvolvido em ASP.NET durante o 4º semestre de minha faculdade. O intuito era desenvolver um software completo web que desenvolvesse algumas funções como sistema, seguindo algumas regras de negócio e que fosse regularmente correspondente com a realidade.

A ideia do Restaurant-e é agilizar os processos de comanda e auxiliar na questão organizacional dentro de um restaurante, provendo diversas funcionalidades para as diferentes funções que são exercidas diariamente em um restaurante.

### Funcionalidades do Sistema

**Observação:** Para entendar mais sobre as funcionalidades recomendo que assista o vídeo disponibilizado ao final do READ

* Relatórios
    * Dashboard
    * Relatórios de Comanda
    * Relatórios de Produto
* Administrador
    * Gerenciar Funcionários
    * Histórico
    * Visualizar Reclamações
* Gerente
    * Gerenciar Cardápio
    * Gerenciar Mesas
    * Registrar Reclamações
* Garçom
    * Seleção de Mesa
    * Visualizar Cardápio
* Cozinha
    * Pedidos Pendentes
* Caixa
    * Pagamentos Pendentes

### Instruções e Demonstração

Para aqueles que não querem executar localmente, este vídeo mostra o software e um pouco de seu planejamento e conceito:
[Apresentação final do projeto](https://youtu.be/vPpbtt40v_Y)

Para executar localmente:

1. Crie um novo banco de dados executando o arquivo "10_12_2020.sql" que está na pasta "banco";
2. Abra a pasta "Restaurante" como projeto dentro do Visual Studio e execute-o;
3. Altere os campos "User id" e "Password" no arquivo Web.config para sua configuração do banco de dados caso seja necessário.
